_ai_index
=========

NON-AUTHORITATIVE helper artifacts for fast navigation (AI + human).
These files are generated. Source of truth remains the repository itself.

Contents:
- _file_map.json: file list + size + sha256
- python_index.json: python symbols/imports + code graph
- entrypoints.json: semantic entrypoints (how to run / where to start)
- _ai_index_INDEX.txt: integrity manifest (sha256 of index outputs)
